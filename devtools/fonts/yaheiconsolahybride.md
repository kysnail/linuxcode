##关于 YaHei Consola Hybrid 字体
`YaHei Consola Hybrid` 字体是我比较喜欢的字体。

 * YaHei.Consolas.1.11 不支持 Windows 7

    XP 下不支持更换 cmd 字体。

 * YaHei.Consolas.1.12 支持 Windows 7

## ClearType
安装完成的 YaHei Consola Hybrid 字体可能会有很强大锯齿感，可以使用 Clear Type 工具消除字体的锯齿感。
