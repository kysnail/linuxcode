脚本基本结构
------------
    Python 脚本的结构非常清晰，在 Python 中不使用 C 语言中的花括号表示语句快，而是用缩进。代码缩进
    一般用在函数定义、类的定义以及一些控制语句中。一般来说，行尾的“:”表示代码缩进的开始。 ::

	if a > b:
	    print a		# 代码缩进，如果 a > b ，则执行 print a
	else:
	    print b		# 代码缩进，如果 a <= b ，则执行 print b
