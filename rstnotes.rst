Tables
======

    制作表格时需要注意，RST 要求表格的三条线的外形完全一致，这点需要注意。::

	=======   =========
	   X	      Y
	=======   =========
	   1          2
	=======   =========

    另外就是，每列数据在 raw 模式下一定要用 tab 进行对齐。

常见错误
========

Title level inconsistent
------------------------
    标题之间的层级关系一定要遵循规定，否则汇报如下错误。::

	SystemMessage: <string>:59: (SEVERE/4) Title level inconsistent:

	demo
	````

    上面这个错误所对应的代码如下：::

	注释种类
	========
	- # -> 单行注释
	- ''' -> 多行注释
	- """ -> 多行注释

	demo
	````
