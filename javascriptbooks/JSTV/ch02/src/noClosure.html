// No matter which node you click, prompt answer is '3'. window.onload = function() { var LiObjects = document.getElementsByTagName('li'); for (var i = 0; i != LiObjects.length; i++) { LiObjects[i].onclick = function() { alert(i); } } } 上面的这种写法，不论点击那个 node 弹出的窗体提示都为 3。因为在发生点击事件时，系统会在当前作用域中查找 i 的值，如果没有则向上查找。这里在查找时变量 i 的值为 3，所以弹出的结果总为 3。
